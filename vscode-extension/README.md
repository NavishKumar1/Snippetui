# SnippetUI

SnippetUI is a premium components library for VS Code. Access hundreds of beautifully animated components designed with Vanilla CSS, JS/TS, and Tailwind CSS.

## Features

- **Hundreds of Premium Components**: From complex text animations to liquid gradient buttons.
- **Tailwind CSS Support**: Choose to output standard HTML/CSS or HTML configured with Tailwind CSS classes.
- **TypeScript Support**: Insert JavaScript or TypeScript interactivity logic.

## Usage

1. Open the command palette (`Ctrl+Shift+P` / `Cmd+Shift+P`).
2. Type `SnippetUI: Insert Component`.
3. Search and select the component you want to insert.
4. Choose your preferred styling framework (Vanilla CSS or Tailwind CSS).
5. Choose your scripting language (JavaScript or TypeScript) if applicable.
6. The component files will be inserted directly into your workspace.

## Extension Settings

This extension contributes the following settings:

* `snippetui.defaultStyling`: Default styling framework ('Prompt', 'Vanilla CSS', 'Tailwind CSS')
* `snippetui.defaultScript`: Default scripting language ('Prompt', 'JavaScript', 'TypeScript')
